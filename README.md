# EnDek

This Python file lets you Encrypt English language with a randomly generated Encryption key. It is a terminal application (as it severves a simple problem, I felt a whole UI was uneccesary)  It provides a config page if you want to change the encryption Key. Users have the ability to add a Encryption key generated by any fellow users or make a random key they can share with their fellow users. First Time you lauch it it will promte you to add a user (feature is still in beta so it is coming up soon...) and set a password to it. but currently the user is set to default my name '''jovancherian''' and the password is set to '''3e3e3e''' you can change it as you wish in the source code. If you don't want to be a user you can still encrypt it by either genarating or entering a encyption key. This comes in handy when you want to just encrypt file the program will NOT allow you to decrypt without have a username (you can use the default username).
